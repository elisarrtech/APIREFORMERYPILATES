# FitnessClub Backend - Sistema de Reservas de Pilates

Sistema backend profesional para gestión de reservas de clases de Pilates.

## Stack Tecnológico

- **Python 3.11+**
- **Flask 3.0.0** - Framework web
- **SQLAlchemy** - ORM
- **Flask-JWT-Extended** - Autenticación JWT
- **Flask-CORS** - CORS
- **Flask-Migrate** - Migraciones de base de datos
- **bcrypt** - Hash de contraseñas

## Características

- ✅ Autenticación y autorización con JWT
- ✅ Sistema de roles (Admin, Instructor, Cliente)
- ✅ Gestión de paquetes de clases
- ✅ Sistema de reservas
- ✅ Panel de administración
- ✅ Panel de instructores
- ✅ API REST completa
- ✅ Validaciones robustas
- ✅ Manejo de errores profesional
- ✅ Logging completo
- ✅ Seguridad bulletproof
